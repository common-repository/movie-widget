Movie widget displays movie posters, trailers and descriptions. It can also do TV shows as well.

== Description ==

Display a movie poster, overview or trailer on your site with just a search. 

Simply type your movie into the widget and your poster is returned. Works with TV shows as well.

Free, fast and multiple poster sizes avaliable all in one widget. 

= Note: =
A TMDB key must be obtained from [TheMovieDB API](https://www.themoviedb.org/faq/api).
This product uses the TMDb API but is not endorsed or officially by TMDb.

== Installation ==

= From your WordPress dashboard =

1. Visit Plugins > Add New
2. Search for 'movie widget'
3. Activate movie widget from your Plugins page. 
4. Obtain a TMDB key from [TheMovieDB API](https://www.themoviedb.org/faq/api)
5. Enter the TMDB key and movie into widget interface 

= From WordPress.org =

1. Download movie widget.
2. Upload the 'movie widget' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate movie widget from your Plugins page. 
4. Obtain a TMDB key from [TheMovieDB API](https://www.themoviedb.org/faq/api) 
5. Enter the TMDB key and movie into widget interface
